Quotes

📖 Descrizione del progetto

"Quotes" è un'app web sviluppata in Angular 19, pensata per offrire un'esperienza interattiva e ispirazionale nella raccolta, visualizzazione e generazione di citazioni. Il progetto unisce funzionalità dinamiche e un’interfaccia fluida, con l'integrazione di librerie moderne e tecnologie intelligenti.

⚙️ Funzionalità principali

📜 Visualizzazione di citazioni consigliate tramite carosello interattivo realizzato con Splide.js ✍️ Aggiunta manuale di citazioni personalizzate, con campi per autore (opzionale) e testo (obbligatorio) 🧠 Generazione automatica di citazioni tramite integrazione con GPT (OpenAI API) → L'utente può richiedere frasi ispirazionali su un tema a scelta 🔍 Searchbar intelligente per il filtraggio dinamico delle citazioni salvate, anche per parole chiave multiple 🧠 Salvataggio persistente delle citazioni personalizzate in localStorage 🗑️ Rimozione e gestione dinamica delle citazioni con aggiornamento automatico dell'interfaccia 📅 Tracciamento della data e ora di salvataggio di ogni citazione 🌍 Interfaccia user-friendly, pensata anche per mobile

💡 Stack e tecnologie utilizzate

Angular 19 – framework principale per struttura e componentizzazione GSAP (GreenSock Animation Platform) – usato per animazioni testuali (effetto scrittura, transizioni dinamiche) Splide.js – libreria per carousel, integrata direttamente in Angular con personalizzazione avanzata (auto scroll, loop, ecc.) OpenAI GPT-3.5-turbo – integrazione via API per generare contenuti testuali ispirazionali RxJS / BehaviorSubject – per gestione reattiva dello stato (savedQuotes$) e aggiornamento in tempo reale localStorage – per la persistenza locale dei dati utente Vercel – per il deploy e hosting del progetto, con gestione delle chiavi API tramite environment variables 🎯 Obiettivo del progetto: L'app nasce con l'intento di fornire uno strumento leggero e creativo per trovare o generare frasi significative da usare in contesti personali, sociali o professionali, mantenendo un'esperienza fluida, interattiva e accessibile. ✅ Cosa dimostra questo progetto: Capacità di integrare API esterne e gestire la sicurezza delle chiavi Uso avanzato di Angular e pattern reattivi Competenze nella gestione dinamica dell'interfaccia e nella user experience Integrazione armoniosa tra design, animazione e funzionalità
