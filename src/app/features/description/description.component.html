Quotes

📖 Descrizione del progetto

Quotes è una web app sviluppata in Angular 19, pensata per offrire un'esperienza interattiva e che permetta la visualizzazione e generazione di citazioni.

I contenuti possono essere inseriti sia manualmente, sia tramite un’API che importa elementi come quelli visualizzati nel carosello di card. Inoltre, è possibile generare automaticamente citazioni grazie all’integrazione con i suggerimenti forniti da OpenAI.

Il progetto unisce funzionalità dinamiche e un’interfaccia responsive, progettata tramite un veloce wireframe, con l'integrazione di librerie moderne e tecnologie intelligenti. Il prototipo è presente in questo link.

⚙️ Funzionalità principali

📜 Visualizzazione di citazioni consigliate tramite carosello interattivo realizzato con Splide.js
✍️ Aggiunta manuale di citazioni personalizzate, con campi per autore (opzionale) e testo (obbligatorio)
🤖 Generazione automatica di citazioni tramite integrazione con GPT (OpenAI API)
💬 L'utente può richiedere frasi ispirazionali su un tema a scelta
🔍 Searchbar intelligente per il filtraggio dinamico delle citazioni salvate, anche per parole chiave multiple
🧠 Salvataggio persistente delle citazioni personalizzate in localStorage (in assenza di un be a cui connettersi)
🗑️ Rimozione e gestione dinamica delle citazioni con aggiornamento automatico dell'interfaccia
📅 Tracciamento della data e ora di salvataggio di ogni citazione
🌍 Interfaccia user-friendly, pensata anche per mobile

💡 Stack e tecnologie utilizzate

Angular 19 – framework principale
GSAP (GreenSock Animation Platform) – usato per animazioni testuali (effetto scrittura, transizioni dinamiche)
Splide.js – libreria per carousel, integrata direttamente in Angular con personalizzazione avanzata (auto scroll, loop, ecc.)
OpenAI GPT-3.5-turbo – integrazione via API per generare contenuti testuali ispirazionali
RxJS / BehaviorSubject – per gestione reattiva dello stato (savedQuotes$) e aggiornamento in tempo reale
localStorage – per la persistenza locale dei dati utente
Vercel – per il deploy e hosting del progetto, con gestione delle chiavi API tramite environment variables
Scss - gestione degli stili con delle mixin, e dividendo tra colori, bottoni, tipografia, rendendoli dinamici riutilizzando ogni foglio tramite una @import nel componente interessato

🎯 Obiettivo del progetto

L'app nasce con l'intento di fornire uno strumento leggero e creativo per trovare o generare frasi significative da usare in contesti personali, sociali o professionali, mantenendo un'esperienza fluida, interattiva e accessibile.
Cosa dimostra questo progetto? Capacità di integrare API esterne e gestire la sicurezza delle chiavi, uso avanzato di Angular e pattern reattivi, competenze nella gestione dinamica dell'interfaccia e nella user experience, integrazione armoniosa tra design, animazione e funzionalità.
